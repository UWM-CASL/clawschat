Browser LLM Runner Help

This page explains core features and the fastest path to complete basic tasks in the app.

Key features

  • Local in-browser chat with model loading and caching.
  • Backend selection: Auto (WebGPU then CPU), WebGPU only, or CPU only.
  • Streaming model responses with a Stop generating control.
  • Conversation history with rename, branch edits, regenerate, and fix actions.
  • Conversation export to Markdown or JSON.
  • Settings for output tokens, context size, temperature, Top K, and Top P.

Basic tasks

  1. Select a model on the setup panel, then choose Load model.
  2. Type in Message input and select Send.
  3. While a response is streaming, select Stop generating to cancel.
  4. Open Settings to change backend and generation controls.
  5. Use actions on model replies to Copy, Regenerate, or Fix.
  6. Use the conversation download menu to export the visible branch.

Accessibility notes

All primary controls are keyboard reachable. Use Tab and Shift+Tab to move focus. Use Escape to close open dialogs and return focus to the trigger.
